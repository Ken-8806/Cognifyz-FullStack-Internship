# Cognifyz Internship - Task 1
## HTML Structure and Basic Server Interaction

### 📌 What this project does
A Student Registration web app built with:
- **HTML + EJS** → Server-side rendered pages
- **Node.js + Express** → Backend server
- **3 Pages** → Registration form, Success page, All Students list

### 🚀 How to Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your browser and go to:
   ```
   http://localhost:3000
   ```

### 📁 Project Structure
```
task1/
├── server.js          → Express server + routes
├── package.json       → Dependencies
└── views/
    ├── index.ejs      → Registration form
    ├── success.ejs    → Success page after registration
    └── students.ejs   → List of all registered students
```
